# Python Development Internship - Beginner Level Tasks

This repository contains the **Beginner Level tasks** completed as part of the **Python Development Internship at ShadowFox**.  
These tasks focus on building a strong foundation in Python programming by implementing basic concepts and problem-solving exercises.

---

## 📌 Topics Covered

- Variables and Data Types
- Numbers and Mathematical Calculations
- Lists and List Operations
- Conditional Statements (if-else)
- Loops (for loop)
- Dictionaries and Data Handling

---

## 📂 Task Files

### 1️⃣ Variables
- `variables_pi_datatype.py` – Store value of pi and check its data type.
- `reserved_keyword_example.py` – Demonstrates why Python keywords cannot be used as variable names.
- `simple_interest_calculator.py` – Calculates Simple Interest using Python variables.

### 2️⃣ Numbers
- `format_function_example.py` – Uses Python `format()` function.
- `circle_area_water_calculation.py` – Calculates area of a circular pond and water volume.
- `speed_calculation.py` – Calculates speed using distance and time.

### 3️⃣ Lists
- `justice_league_list_operations.py` – Performs operations like adding, removing, sorting, and updating list elements.

### 4️⃣ If Conditions
- `bmi_category_checker.py` – Determines BMI category based on user input.
- `city_country_identifier.py` – Identifies which country a city belongs to.
- `same_country_checker.py` – Checks whether two cities belong to the same country.

### 5️⃣ For Loop
- `dice_roll_simulation.py` – Simulates rolling a dice multiple times.
- `jumping_jacks_workout.py` – Interactive workout counter using loops and conditions.

### 6️⃣ Dictionary
- `friends_name_length_tuple.py` – Creates tuples containing names and their lengths.
- `trip_expense_tracker.py` – Tracks and compares travel expenses using dictionaries.
